Numeric Operators

Implies, > (IsLarger), >= (IsLargerEqual), < (IsSmaller), <= (IsSmallerEqual), = (IsEqual), <> (IsNotEqual), + (Add), - (Subtract), or, * (Multiply), / (Divide), ^ (Power), and, mod, not*.

Numeric Constants

ER, False, NA, No, Off, On, PM, True, Yes, Title*, NoTrend, Trend, User, Sector, Period constants, Single, Detail.

Outdated: None, N, Y.

Other constants in the FES are: BKNPERIOD, SECTORPERIOD, YEAR, BOOKYEAR, BACKWARD, COLUMN, NONE

All Numeric Constants have to be written with UPPERCASES while using it with the FES 2.0.0 and higher
